# golang-toolkit

Golang Toolkit

- 使用

  ``go get -u github.com/acexy/golang-toolkit``

封装常用工具或者第三方依赖

- 提供处理类型类型
  - 内存缓存
  - 加密
  - 算法
  - 工具

- 第三方库清单

| 库                            | 类型          |
|------------------------------|-------------|
| github.com/go-resty/resty/v2 | http client |
| github.com/sirupsen/logrus   | logger      |
| github.com/allegro/bigcache  | cache |
| gopkg.in/gomail.v2           | email |
